## How To Install

1. Open the link on Android, preferably in Chrome.
2. Download the APK.
3. If Android asks, allow installation from Chrome.
4. If Google Play Protect warns, tap details/more and choose install anyway only if you trust this beta.
5. Android 7.0 or newer is recommended.
